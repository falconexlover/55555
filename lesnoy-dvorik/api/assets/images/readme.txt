1. logo.png - Логотип гостиницы "Лесной дворик" (рекомендуемый размер 150x50 пикселей)
2. hotel-main.jpg - Главное изображение гостиницы для баннера (рекомендуемый размер 1920x800 пикселей)
3. about.jpg - Изображение для раздела "О нас" (рекомендуемый размер 600x400 пикселей)
4. romantic.jpg - Изображение для спецпредложения "Романтический уик-энд" (рекомендуемый размер 400x300 пикселей)
5. family.jpg - Изображение для спецпредложения "Семейный отдых" (рекомендуемый размер 400x300 пикселей)
6. family-offer.jpg - Дополнительное изображение для семейного отдыха (рекомендуемый размер 400x300 пикселей)
7. business.jpg - Изображение для бизнес-пакета (рекомендуемый размер 400x300 пикселей)
8. weekend.jpg - Изображение для предложения "Выходные в лесу" (рекомендуемый размер 400x300 пикселей)
9. lux-package.jpg - Изображение для люкс-пакета (рекомендуемый размер 400x300 пикселей)
10. spring.jpg, summer.jpg, autumn.jpg, winter.jpg - Изображения для сезонных предложений (рекомендуемый размер 300x200 пикселей)
11. gift-certificate.jpg - Изображение для подарочного сертификата (рекомендуемый размер 400x300 пикселей)
12. map-marker.png - Иконка для Яндекс.Карты (рекомендуемый размер 40x40 пикселей)

Если у вас нет подходящих изображений, вы можете:

1. Использовать стоковые фотографии с бесплатных ресурсов, таких как:
   - Unsplash (https://unsplash.com/)
   - Pexels (https://www.pexels.com/)
   - Pixabay (https://pixabay.com/)

2. Создать временные заглушки на сервисах:
   - Placeholder.com (например: https://via.placeholder.com/400x300/5a8f7b/ffffff?text=Лесной+дворик)
   - DummyImage (https://dummyimage.com/)

ВАЖНО: При загрузке изображений убедитесь, что имена файлов совпадают с указанными выше, иначе они не будут отображаться на сайте. Также убедитесь, что все изображения оптимизированы для веб (сжаты) для ускорения загрузки страниц.
